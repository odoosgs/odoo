Features

This module brings the power of the Partner Ledger report to your customer portal. Perfect for businesses that want to provide self-service account statements to their customers, reducing support requests and improving customer satisfaction.

✓ New "Customer Statement" menu in the customer portal

✓ Filter by Month, Quarter, Year, or Custom Date Range

✓ Initial balance calculation for filtered periods

✓ Running balance for each transaction line

✓ Negative balances highlighted in red for easy identification

✓ Export to PDF with professional formatting

✓ Secure: customers only see their own data

How It Works

1

Install Module

Install the module from Apps. It automatically adds a "Customer Statement" entry to the portal home page for all portal users.

2

Customer Logs In

Your customer logs into the portal and clicks on "Customer Statement" to view their account transactions.

3

View & Export

Customer can filter by date, view their running balance, and export the statement to PDF for their records.
