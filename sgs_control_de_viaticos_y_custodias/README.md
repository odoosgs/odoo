# Módulo Personalizado para Odoo 19.0: SGS Control de Viáticos y Custodias

Módulo generado automáticamente con la plataforma de desarrollo interactivo Odoo 19 SDK Scriptor.

## Instrucciones de Instalación

1. Descarga y extrae este archivo ZIP dentro de tu sistema de archivos de Odoo addons (directorio `custom_addons`).
2. Verifica que el nombre de la carpeta sea exactamente `sgs_control_de_viaticos_y_custodias`.
3. Reinicia tu servidor de Odoo para que reconozca el nuevo directorio en la terminal:
   ```bash
   ./odoo-bin -c /ruta/a/tu/odoo.conf
   ```
4. Inicia sesión en Odoo 19 con permisos de administrador.
5. Ve a **Ajustes** y activa el **Modo de Desarrollador**.
6. Ve al menú **Aplicaciones** y haz clic en **Actualizar lista de aplicaciones**.
7. Busca tu módulo por su nombre `SGS Control de Viáticos y Custodias` o `sgs_control_de_viaticos_y_custodias`.
8. Haz clic en **Instalar** (o Activar) para ver las nuevas columnas y flujos de trabajo aplicados.

## Dependencias
Este módulo depende de:
- `base`
- `web`
- `mail`
- `portal`
- `hr`
